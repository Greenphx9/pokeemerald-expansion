# fire-red

## What is this?
`fire-red` is a branch that aims to port all FRLG to pokeemerald. Specifically, pokeemerald-expansion. I am attempting to keep both Emerald & FRLG features in the game. For example, the branch currently supports both Emerald & FRLG tileset formats. Connections do not work, but you can warp from an Emerald map to a FRLG map without any huge problems.
